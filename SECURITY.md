# Security Policy

## Personal project

This is a personal project.

Do not depend on this project.

In particular, do not depend on this project its security.

There is no good mechanism configured for security updates.

There is no support available.

## Reporting a vulnerability

Please use the "Report a vulnerability" button under the Security tab of the github project.

If you cannot use GitHub's vulnerability reporting workflow, please send e-mail to security @ leosimons dot com.
